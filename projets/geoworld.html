Projet GeoWorld

Réalisé en première année de BTS SIO SLAM - 2023-2024

Présentation du projet

GeoWorld est une application web permettant d'afficher et de consulter des informations géopolitiques et économiques des pays du monde entier. Les données sont stockées dans une base de données SQL et proviennent de l'Institut Official Statistics of Finland (2006).

L'application permet de visualiser les pays par continent, d'afficher leurs capitales, drapeaux et diverses informations géopolitiques.

Fonctionnalités implémentées

  • Affichage des pays regroupés par continent
  • Menu déroulant (dropdown) pour sélectionner un continent spécifique
  • Affichage des capitales de chaque pays
  • Affichage des IDs des continents
  • Intégration des drapeaux pour chaque pays

Étapes de développement

1. Affichage des informations par pays

J'ai commencé par implémenter la fonction getCountriesByContinent qui récupère toutes les informations concernant les pays depuis la base de données SQL.

2. Utilisation d'une boucle foreach pour l'affichage

Pour afficher ces informations sur le site, j'ai utilisé une boucle foreach avec une condition if/else pour traiter les données de chaque pays par continent.

Architecture technique

Structure de la base de données

Le projet GeoWorld s'appuie sur une base de données relationnelle comportant plusieurs tables interconnectées:

  • Country: Contient les informations principales sur les pays (nom, continent, population, etc.)
  • City: Stocke les données des villes, avec une référence au pays correspondant
  • Language: Répertorie les différentes langues
  • CountryLanguage: Table de liaison entre pays et langues, indiquant les langues parlées dans chaque pays
